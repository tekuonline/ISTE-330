Login Screen

When you open the application of Research paper this is the main screen which appears as shown below.

To login as Student, Faculty, Admin you need to click on the login button which is highlighted in the image below.


After clicking on the Login button the following below login screen appears.

  • Enter username and password credentials to login as a student/faculty/admin.
  • If user authentication successful next screen will appear depending on the role of user.
  • If user authentication failed it will give an error message asking to check credentials.
