About

Department faculty regularly engage in research and publish their results. They are often looking to work with other faculty and students, but it is difficult to keep track of what each person is working on .Research Paper Database is a database application which allows the users to search the papers in a database.  Depending on the type of user access is given to add, modify, update and delete the papers in the applications. It is possible to retrieve the association and owner of the paper through which it allows a student and faculty to work together. It has a secured level of authorization which allows only authenticated users to perform operations like add, modify and delete papers from the system.
